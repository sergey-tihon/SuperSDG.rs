# SuperSDG 3: The Maze Game

Rust edition of my very first maze game initially created in Pascal
then [ported to 3d (C++ & OpenGL)](https://sergeytihon.com/2013/03/16/supersdg2-the-maze-game-—Å-opengl/).

## Development

Run locally in the native window:

```shell
cargo run
```

Run locally in the browser:

```shell
trunk serve
```
